Project Name: Aura Dashboard
Version: 1.0.0
Built With: React, TypeScript, Vite

--------------------------------------------------------------------------------
                              PROJECT OVERVIEW
--------------------------------------------------------------------------------

Aura Dashboard is a full-featured social media dashboard application built with
modern web technologies. The application provides a comprehensive platform for
social networking interactions, featuring real-time-like updates, responsive
design, and an intuitive user interface.

Key Capabilities:
  • User profile management with customizable information
  • Interactive post feed with like and comment functionality
  • Comprehensive notifications system
  • Real-time messaging interface
  • Content discovery through explore features

--------------------------------------------------------------------------------
                           TECHNOLOGY STACK
--------------------------------------------------------------------------------

Frontend Framework:
  • React 18.3.1 - Core UI library for component-based architecture
  • TypeScript 5.8.3 - Type-safe JavaScript replacement (chosen over 
    JavaScript during development for better type safety and developer 
    experience while learning)

Build Tools:
  • Vite 5.4.19 - Next-generation frontend build tool
  • ESLint 9.32.0 - Code linting for maintaining quality

Styling & UI:
  • Tailwind CSS 3.4.17 - Utility-first CSS framework
  • Radix UI - Accessible component primitives
  • Framer Motion 12.34.3 - Animation library
  • Lucide React - Icon library
  • Class Variance Authority - Component variant management
  • Tailwind Merge & CLSX - Utility for merging Tailwind classes

State Management & Data:
  • TanStack React Query 5.83.0 - Server state management
  • React Hook Form 7.61.1 - Form handling
  • Zod 3.25.76 - Schema validation

Routing:
  • React Router DOM 6.30.1 - Client-side routing

Additional Libraries:
  • Recharts 2.15.4 - Charting library
  • Embla Carousel - Touch-friendly carousel
  • Date-fns 3.6.0 - Date manipulation
  • Next Themes 0.3.0 - Theme management support

Testing:
  • Vitest 3.2.4 - Unit testing framework
  • Testing Library React 16.0.0 - Component testing

Component Architecture:
  The application follows a modular component-based architecture where:
  • UI components are built using Radix UI primitives
  • Feature components encapsulate specific functionality
  • Pages serve as route-level containers
  • Layout components provide consistent structure

--------------------------------------------------------------------------------
                         CORE FEATURES
--------------------------------------------------------------------------------

1. User Profile Section
   • Profile display with avatar, bio, and stats
   • Edit profile functionality
   • Posts, followers, and following counts
   • Liked posts gallery
   • Animated stat cards with gradient backgrounds

2. Post Feed with Like/Comment Functionality
   • Create new posts with text content
   • Image support in posts
   • Like/unlike posts with animated feedback
   • Expandable comment sections
   • Share and bookmark options
   • Real-time-like updates with optimistic UI

3. Notifications Panel
   • Multiple notification types (likes, comments, follows, mentions)
   • Unread notification indicators
   • Animated notification entries
   • Mark as read functionality

4. Message System
   • Conversation list with unread counts
   • Real-time chat interface
   • Message timestamps
   • User avatars and status indicators

5. Content Discovery (Explore)
   • Trending topics display
   • Suggested users to follow
   • Search functionality in navbar

6. Responsive Design
   • Mobile-first approach
   • Collapsible sidebar for mobile
   • Adaptive layouts for all screen sizes
   • Touch-friendly interactions

7. Dark/Light Mode Support
   • Theme toggle capability
   • System theme detection
   • Consistent styling across themes

8. Search Functionality
   • Global search in navbar
   • Search state management
